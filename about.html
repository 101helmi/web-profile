Halo, Saya Helmi Estiawan

Saya adalah seorang Backend Developer dengan passion dalam membangun solusi digital yang efisien dan scalable. Memiliki ketertarikan khusus pada Python, Machine Learning, dan pengembangan aplikasi web.

Perjalanan saya di dunia programming dimulai dari keingintahuan akan bagaimana teknologi dapat memecahkan masalah sehari-hari. Sejak saat itu, saya terus belajar dan mengembangkan skill untuk menjadi developer yang lebih baik.

Saat ini, saya fokus pada pengembangan backend systems, data processing, dan mengeksplorasi dunia Machine Learning. Saya percaya bahwa teknologi memiliki kekuatan untuk membuat dampak positif di masyarakat.
