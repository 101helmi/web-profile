Tomato Maturity Classification

2024
Machine Learning / Deep Learning
Skripsi / Research
View on GitHub

Overview

Project ini merupakan penelitian skripsi yang mengembangkan sistem klasifikasi tingkat kematangan tomat menggunakan metode Deep Learning dan Object Detection. Sistem dapat mendeteksi dan mengklasifikasikan tomat ke dalam beberapa kategori kematangan secara real-time.

Background

Dalam industri pertanian, penentuan tingkat kematangan buah secara manual membutuhkan waktu dan tenaga yang besar. Kesalahan dalam menentukan kematangan dapat menyebabkan kerugian ekonomi. Sistem otomatis berbasis AI dapat membantu petani dan distributor untuk sorting buah dengan lebih efisien dan akurat.

Methodology

  • Pengumpulan dataset gambar tomat dengan berbagai tingkat kematangan
  • Data augmentation untuk meningkatkan variasi dataset
  • Training model menggunakan YOLO untuk object detection
  • Fine-tuning dengan transfer learning dari pre-trained model
  • Deployment ke aplikasi Android untuk penggunaan praktis

Tech Stack

Python TensorFlow YOLO OpenCV Android Kotlin NumPy Matplotlib

Results

Model yang dikembangkan mencapai akurasi tinggi dalam mengklasifikasikan tingkat kematangan tomat:

  • Akurasi overall: ~92%
  • Dapat mendeteksi multiple tomat dalam satu frame
  • Real-time inference pada perangkat Android
  • Mampu membedakan 4 tingkat kematangan: Mentah, Setengah Matang, Matang, Terlalu Matang

Challenges & Learnings

  • Handling variasi pencahayaan dalam gambar
  • Optimasi model untuk device mobile dengan resource terbatas
  • Balancing antara accuracy dan inference speed
  • Pengumpulan dan labeling dataset yang berkualitas
