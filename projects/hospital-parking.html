Hospital Parking System

2022 - 2023
Full Stack Web Development
Internship Project
View on GitHub

Overview

Sistem manajemen parkir terintegrasi untuk rumah sakit yang memungkinkan monitoring real-time ketersediaan slot parkir, pencatatan kendaraan masuk/keluar, dan perhitungan biaya parkir otomatis. Dikembangkan selama periode internship sebagai bagian dari tim development.

Problem

Rumah sakit menghadapi beberapa masalah dengan sistem parkir manual:

  • Tidak ada informasi real-time tentang ketersediaan slot parkir
  • Antrian panjang di pintu masuk dan keluar
  • Kesulitan dalam menghitung durasi dan biaya parkir
  • Tidak ada data untuk analisis penggunaan parkir

Solution

Mengembangkan sistem parkir digital dengan fitur:

  • Dashboard real-time untuk monitoring slot parkir
  • Sistem barcode/QR untuk entry dan exit kendaraan
  • Perhitungan otomatis durasi dan biaya parkir
  • Laporan harian, mingguan, dan bulanan
  • Integrasi dengan printer struk
  • Kategori parkir berbeda untuk staff dan visitor

Tech Stack

CodeIgniter 3 PHP MySQL Bootstrap 4 jQuery AJAX Chart.js

My Role

Sebagai intern developer, kontribusi saya meliputi:

  • Mengembangkan modul laporan dan statistik
  • Implementasi fitur real-time update menggunakan AJAX
  • Membuat interface dashboard dengan Chart.js
  • Bug fixing dan optimization
  • Dokumentasi teknis
